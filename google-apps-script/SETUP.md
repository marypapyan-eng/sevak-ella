# Подключение RSVP к Google Таблице

Таблица: [Свадьба – ответы](https://docs.google.com/spreadsheets/d/13K_kmKb7-DkNshI1zJYg1BR0T5DfFcOrncMQUhCBxYY/edit)

## 1. Установить скрипт

1. Откройте таблицу (нужны права **редактора**).
2. **Расширения** → **Apps Script**.
3. Удалите код по умолчанию и вставьте содержимое файла `Code.gs` из этой папки.
4. Сохраните (Ctrl/Cmd + S).

## 2. Развернуть веб-приложение

1. **Развернуть** → **Новое развертывание**.
2. Тип: **Веб-приложение**.
3. **Выполнять от имени:** я (владелец таблицы).
4. **У кого есть доступ:** все.
5. Нажмите **Развернуть** и разрешите доступ к таблице.
6. Скопируйте **URL веб-приложения** (вид `https://script.google.com/macros/s/.../exec`).

## 3. Вставить URL на сайт

Откройте `js/rsvp-config.js` и вставьте URL:

```js
window.RSVP_CONFIG = {
  scriptUrl: 'https://script.google.com/macros/s/ВАШ_ID/exec',
};
```

## 4. Проверка

1. Откройте сайт приглашения, заполните форму, нажмите **Отправить**.
2. В таблице должна появиться строка: дата, ответ, имя.
3. Если ошибка — в Apps Script откройте **Выполнения** и посмотрите лог.
